写在2016年的最后一天 - 我的第一年工作总结

2016年即将结束，转眼间，这已经是我参加工作的第七个月了。看到园子里很多人都写了年终总结，我也想写点什么，来纪念我的2016。

工作

2016年4月底，我毕业了。并没有多么激动的情绪，也没有像大多数人那样发说说、发朋友圈。就这样平平淡淡的毕业了，参加了工作，步入了社会。​

2016年5月2日，我来到了北京。记得那天是早上到的，下着小雨。我和一帮同学一起，在达官营那边临时住了下来。休息了一天之后，就开始出去面试了。经过一个多星期的面试，5月10日，我找到了人生中的第一份工作。

参加工作后，接触了很多新的技术。很多东西都是从头开始学起。下面是我今年所学到一些技术。

  • Spring
  • Sping mvc
  • MyBatis
  • Jquery
  • Bootstrap
  • Vue
  • Spring Data Jpa
  • SVN
  • Git

虽然学了这么多技术，但大多还只是停留在会用的程度。

说来惭愧，买了一本《Java核心技术 卷一》，直到现在还没看完。执行力还是太差！

生活

找到工作之后，我和几位大学同学一起，重新租了房子。房子还不错，就是路上灰尘太多了。

刚开始的几个月，我们去了很多北京的景点。我们去了长城、故宫、颐和园、北京动物园、香山以及一些公园。后来，天气慢慢转冷之后，出去的就少了。

本来小区附近的一个工厂有篮球场，我还特意去买了个篮球，每个周六或周日下午，我都会去打会篮球。但后来那个工厂不让人进去打了。不能打篮球，我就和我同学去买了羽毛球，但北京经常刮风，在室外根本就打不了，于是也只有作罢。现在明显感觉到身体素质没以前好了，上个楼梯都上气不接下气的。过完年后，可能的话，我想去附近的健身房办张健身卡。

日子本来就在平淡中慢慢地过着。但11月底，生活跟我开了一个很大的玩笑。在武汉工作的父亲发生了车祸，当时严重到都送去抢救了。庆幸的是，现在父亲人已经没事了，只是有点说胡话。经过这次事故，我更加体会到家人的重要性。

总结

2016年，开始了新的生活，也学了很多工作技能。但，还是浪费了很多时间在没有意义的事情身上。现在正是奋斗的年纪，不要选择安逸。

再过几十分钟，就是2017年了。新的一年，祝愿家人朋友平平安安、健健康康。希望自己踏踏实实学技术、认认真真过日子。
